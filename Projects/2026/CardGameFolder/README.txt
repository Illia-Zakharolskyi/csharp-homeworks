# CardGame - карточна RPG гра.

CardGame — це консольна RPG-гра, де гравець обирає персонажа та вступає в покрокову битву з випадковим ворогом, завдаючи збиток картками, які випали з спеціальної колоди вибранного персонажа.
Гра використовує JSON-файли для зберігання карт, характеристик та списків персонажів.

## Автор 
Розробник: Ілля 
Мова програмування: C# (.NET 9.0) 
Тип проєкту: Консольна гра

## Основні можливості
- Вибір персонажа з меню
- Збереження вибору персонажа у AppData
- Генерація випадкового ворога
- Завантаження карт та характеристик з JSON-файлів
- Покрокова бойова система
- Форматування карт у зручному вигляді

## Як запустити проєкт
1. Відкрити проєкт у Visual Studio або JetBrains Rider
2. Переконатися, що встановлено .NET 9.0
3. Запустити проєкт у режимі Debug або Release

## Як запустити .exe
> Увага: `bin/` та `obj/` не включені до репозиторію. Для запуску проєкту потрібно зібрати його локально.
1. Перейти до папки: bin/Release/net9.0/win-x64/publish/
2. Запустити CardGame.exe
3. При першому запуску буде створено файл збереження у:
   %AppData%/CardGame/UserChoices.json

## Техничні Деталі
- Всі шляхи до JSON-файлів будуються через AppContext.BaseDirectory
- Використовується власний PathResolver для коректної роботи .exe
- Дані персонажів та карт завантажуються через JsonLoader
- Збереження користувача зберігається у AppData, що дозволяє мати окремі налаштування на кожному пристрої
